# Project Brief: QA Agent Skills Ecosystem

## Project Overview

**Project:** QA Agent Skills Ecosystem  
**Type:** Cursor agent skills collection for QA automation  
**Scope:** Full QA lifecycle coverage through specialized, composable skills

## Goal

Deliver **45+ Cursor agent skills** that cover the complete QA lifecycle:

- **Documentation generation** — requirements, specs, traceability
- **Test case design** — techniques, coverage, risk-based
- **Test writing** — TypeScript + Python matrix (Jest, Vitest, Playwright, pytest, etc.)
- **Reporting & analysis** — results, metrics, insights
- **Orchestration** — workflows, handoffs, automation chains

## Core Capabilities

| Domain | Capabilities |
|--------|--------------|
| Documentation | Requirements docs, test plans, traceability matrices |
| Design | Test case design, coverage analysis, risk assessment |
| Implementation | TS/Python tests across frameworks (Playwright, Jest, pytest, etc.) |
| Reporting | Results parsing, metrics, dashboards |
| Orchestration | End-to-end flows, skill chaining, automation |

## Power Stack

Powered by **6 MCP servers**:

- **Playwright** — browser automation, record/replay
- **GitHub** — repos, PRs, issues, CI
- **Filesystem** — file ops, templates, artifacts
- **Sequential Thinking** — stepwise reasoning
- **Memory** — context persistence
- **Context7** — extended context retrieval

| Standard | Purpose |

## Success Criteria

- Skills are independent, reusable, and orchestrable
- Full coverage of QA lifecycle stages
- Standards-compliant outputs
- Multi-editor support (Cursor-first, then VS Code, Claude Code, Windsurf)
